# 硬件清单文档

## 1. 主控板 — ESP32-S3 开发板

### 模块型号
- **模组**: ESP32-S3-WROOM-1-**N16R8**（16MB Flash，8MB OPI PSRAM）
- **USB 转串口芯片**: CH343P

### 芯片规格（ESP32-S3）
| 参数 | 值 |
|------|-----|
| CPU | 32-bit Xtensa® 双核 @ 240MHz |
| Wi-Fi | IEEE 802.11 b/g/n 2.4GHz |
| 蓝牙 | BLE 5 Mesh |
| SRAM | 512KB（含 16KB RTC SRAM） |
| ROM | 384KB |
| GPIO | 45 个 |
| 外设 | 4x SPI, 3x UART, 2x I2C, 2x I2S, 14x Touch, RMT, LED PWM, USB-OTG, 2x 12-bit ADC, LCD 接口, DVP |

### 模组存储规格对比
| 型号 | PSRAM | Flash |
|------|-------|-------|
| N8R2 | 2MB | 8MB |
| N8R8 | 8MB | 8MB |
| N16R8 | 8MB | 16MB |

### 板载外设
| 外设 | 描述 |
|------|------|
| RGB LED | WS2812 |
| 按键 | RST 按键、BOOT 按键 |
| 指示灯 | PWR 灯、TX 灯、RX 灯 |
| USB 接口 1 | ESP32-S3 直通 TYPE-C（USB & OTG） |
| USB 接口 2 | USB 转串口 TYPE-C（CH343P） |

### 供电方式
1. USB 转 UART 接口 或 ESP32-S3 USB 接口供电（推荐，可同时接）
2. 5V 与 GND 排针供电
3. 3V3 与 GND 排针供电

## 2. 音频放大器模块 — MAX98357A

### 基本信息
| 参数 | 值 |
|------|-----|
| 芯片 | MAX98357A |
| 接口 | I2S 数字音频 |
| 支持格式 | 35 种时钟 / 128 种数字音频格式 |
| 兼容平台 | 树莓派、Arduino、ESP32 等 |

### 引脚定义（排针侧，共 7 针）
| 编号 | 引脚名 | 说明 |
|------|--------|------|
| 1 | VIN | 电源输入 |
| 2 | GND | 电源地 |
| 3 | SD | Shutdown（低电平关闭输出） |
| 4 | GAIN | 增益控制 |
| 5 | DIN | I2S 数字音频数据输入 |
| 6 | BCLK | I2S 位时钟 |
| 7 | LRC | I2S 左右声道时钟（LRCLK/WS） |

### 扬声器接线端
- 左侧绿色接线柱：接扬声器正（+）和负（−）

## 3. 扬声器

### 规格
| 参数 | 值 |
|------|-----|
| 型号 | 2828-4Ω2W L120MM 2P2.54 |
| 尺寸 | 28 × 28mm |
| 阻抗 | 4Ω |
| 额定功率 | 2W |
| 导线长度 | 120mm |
| 连接器 | 2P 2.54mm 间距插头 |

### 接线
- 红线：正极（+）
- 黑线：负极（−）
- 接至 MAX98357A 模块的扬声器接线柱

## 4. OLED 显示屏

### 基本信息
| 参数 | 值 |
|------|-----|
| 通信接口 | I2C |
| 供电电压 | 3.3V ～ 5V |

### 引脚定义（4 针）
| 引脚 | 说明 |
|------|------|
| VCC | 电源正（3.3V 或 5V） |
| GND | 电源地 |
| SCL | I2C 时钟线 |
| SDA | I2C 数据线 |

> **注意**: 该模块有两种引脚排列顺序——VCC 开头（VCC-GND-SCL-SDA）和 GND 开头（GND-VCC-SCL-SDA），接线前请核对实物丝印。

---

## 5. 八位独立按键模块

### 基本信息
| 参数 | 值 |
|------|-----|
| 按键数量 | 8 个独立按键（K1～K8）+ 1 个 POWER 按键（K9） |
| 板载指示灯 | 1 个绿色 LED（电源指示） |

### 引脚定义（排针，10 针）
| 引脚 | 说明 |
|------|------|
| VCC | 电源正 |
| GND | 电源地 |
| K1 | 按键 1 信号（红色） |
| K2 | 按键 2 信号（黄色） |
| K3 | 按键 3 信号（蓝色） |
| K4 | 按键 4 信号（绿色） |
| K5 | 按键 5 信号（黄色） |
| K6 | 按键 6 信号（深色） |
| K7 | 按键 7 信号（白色） |
| K8 | 按键 8 信号（POWER 区） |

> K9（POWER 按键）在板上独立，通过板子连接至 VCC/GND，不单独引出信号脚。
